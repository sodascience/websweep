Contains details for each url found by the scraper

Columns, detail
1, ID number (KvK in our example)
2, URL domain (excluding www)
3, Level at which the page was found
4, URL
5, Status code: 
    - "-9" if the URL was excluded by the classifier
    - An HTML response code
    - A Python error code if an error was encountered (e.g. [SSL] internal error)
6, Date at which the url was seen
7, Relative path at which the url was downloaded ("" if it was not downloaded)
